# 📊 项目进度管理 (Project Tracker)

一个纯 HTML 的单页项目进度管理应用，零外部依赖，打开即用。

## ✨ 功能

- **项目管理** — 新建、编辑、复制、删除项目
- **字段完整** — 品牌、项目名、状态、待办、里程碑、客户对接人、日期、备注（支持粘贴图片）
- **双视图** — 表格视图 ↔ 统计图表（环形图 + 柱状图 + 时间线）
- **数据持久化** — 自动保存到浏览器 localStorage
- **导入导出** — JSON 备份、真正的 .xlsx Excel 导出（多 Sheet）
- **智能提醒** — 逾期/临期里程碑高亮 + 浏览器桌面通知
- **右侧详情面板** — 点击项目展开完整信息，支持直接勾选待办和里程碑
- **快捷键** — Ctrl+N 新建、Ctrl+S 保存、Esc 关闭

## 🚀 使用

直接用浏览器打开 `index.html` 即可。

```
open index.html
```

## 📦 技术

- 纯 HTML/CSS/JS，零外部依赖
- Canvas 2D 手绘图表
- 自建 ZIP + XLSX 生成器
- 暖色珊瑚粉设计风格
